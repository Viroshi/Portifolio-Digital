Propósito do Projeto

Este portfólio consolida as explorações práticas e teóricas desenvolvidas ao longo do semestre. O objetivo primário é demonstrar a fluência na manipulação de diferentes linguagens visuais digitais, articulando técnica e intenção artística.

A estrutura abrange três eixos centrais de experimentação:

  • Pixel Art: A economia de recursos e a precisão do controle individual de pixels, remetendo às origens das interfaces gráficas.
  • Glitch Art: A estética do erro, corrompendo intencionalmente dados para revelar a materialidade frágil do meio digital.
  • CSS Art: O desenho gerativo e estrutural, utilizando código de estilização não apenas para formatar, mas para criar composições visuais complexas.
