# PROJETO PORTIFÓLIO

Site desenvolvido para apresentação da discipla de Eletiva do Curso de Tecnologia em Sistemas para Internet.

Tecnologias usadas:
- Google Stitch para desing Front-end;
- ChatGPT para animações visuais;
- Tailwind CSS para melhor estilização;

Passo a passo para executar o site manualmente:
- Baixe o arquivo ZIP em "Code" > Download ZIP;
- Extraia o arquivo e execute o arquivo index.html que o site funcionará normalmente;

Para executar na web basta acessar o link: https://viroshi.github.io/Portifolio-Digital/
